# Case presentation tamplate

**version 0.0.1**

An effort to put together the r codes that public health specialists frequently need to analyze their data.

## Contributor

- Visnu Pritom Chowdhury <visnu.pritom@icddrb.org>

## Prerequisites

- [x] r
- [x] r editor (e.g. rStudio) 
- [x] data

- getting started 
	- set up directory 
	- import data
	- list variable names 
	- view data set 
	- sort by variable 
- plots 
	- boxplot 
	- histogram 
	- density plot 
	- scatter plot 
- t-tests
	- one sample t-test
	- unpaired sample t-test
	- paired sample t-test (incomplete)
- testing normality
	- with histogram 
	- with qq plot  
	- with stem and leaf plot 
	- with boxplot
	- with skewness
	- with kurtosis
	- shapiro-wilk test
- chi-square test
	- goodness of fit
	- test of independence
	- test of homogeneity
- fisher's exact test
- merging data sets
